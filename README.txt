Zur Ausführung wird Java Runtime Environment 7 und eine aktive Netzwerkverbindung benötigt.

Die beiliegende JAR-Datei "ChatterBox.jar" wird über die Kommandozeile mittels Befehl "java -jar ChatterBox.jar" gestartet und öffnet eine graphische Benutzeroberfläche.
Alle benötigten Libraries/Dateien sind in dieser lauffähigen JAR-Datei enthalten.

Um das Programm mehrmals auf dem selben Computer zu starten, muss die JAR-Datei "ChatterBox.jar" zusätzlich in ein anderes Verzeichnis kopiert werden, da jeweils ein Konfigurationsprofil im Verzeichnis,
von dem aus das Programm gestartet wird, angelegt wird.

Nachdem das Programm gestartet wurde erscheint ein Fenster mit 2 Textfeldern. In "Nickname" wird der später im Chat angezeigte Name angegeben. Das Feld "Gruppenkennung" gibt den Chatraum an,
dem über den Button "Anmelden" beigetreten werden soll. Benutzer die die gleiche Gruppenkennung wählen, gelangen in den selben Chatraum und können nun miteinander über das Textfeld "Message" chatten.
